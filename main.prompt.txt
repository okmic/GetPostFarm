ИНСТРУКЦИЯ ДЛЯ ПРОМТА — ДИЗАЙН-СИСТЕМА ПРОЕКТА POSTFARM
ВСЕ НА TYPESCRIPT REACT TAILWIND, КОД без комментариев и ;
========================================================

1. БРЕНД-КОНЦЕПЦИЯ
------------------
Название: ПостФерма / PostFarm
Слоган: Выращиваем контент. Автоматически. / Grow your content. Automatically.
Ключевые слова: ферма, контент, автоматизация, постинг, синхронизация, выращивание, урожай
Тон общения: дружелюбный, аграрно-технологичный, простота и забота


2. ЦВЕТОВАЯ ПАЛЕТРА
-------------------
Основные цвета (брендовые):
- FarmNight: #1E3A1E (тёмно-зелёный) — фон, основные элементы
- FarmSpark: #4CAF50 (сочный зелёный) — акценты, кнопки, ссылки
- FarmGold: #FFA726 (золотой/янтарный) — урожай, успех, выделение

Нейтральные цвета:
- Белый: #FFFFFF — основной текст, иконки
- Серый светлый: #F5F7F5 — карточки на тёмном фоне
- Серый тёмный: #2C3E2C — второстепенный текст
- Серый холодный: #7F8D7F — подсказки, плейсхолдеры

Статусные цвета:
- Успех: #4CAF50 (зелёный)
- Ошибка: #E57373 (тёплый красный)
- Внимание: #FFA726 (янтарный)

Прозрачности:
- Текст второстепенный: white/0.4 или white/0.6
- Фон карточек: white/0.05 или white/0.08
- Границы: white/0.1 или white/0.15


3. ФОНЫ И ПОВЕРХНОСТИ
---------------------
Основной фон: #1E3A1E (сплошной)

Альтернативные фоны:
- Карточки: #1E3A1E + white/0.05 (blur 10px)
- Модальные окна: #1E3A1E + white/0.08 (blur 20px)
- Выделенные блоки: градиент #1E3A1E → #4CAF50 / 0.1

Фоновый паттерн (опционально):
- Микро-сетка из точек (размер 1px, отступ 24px, цвет white/0.05)
- Точки случайно загораются цветом FarmGold (анимация 4 секунды)

Градиенты (для кнопок и акцентов):
- Градиент 1: #1E3A1E → #4CAF50 (от ночи к росту)
- Градиент 2: #4CAF50 → #FFA726 (от роста к урожаю)
- Градиент 3: #1E3A1E → #FFA726 (от ночи к рассвету)


4. ШРИФТОВАЯ СИСТЕМА
--------------------
Основной шрифт: Inter (кириллица и латиница)
- Заголовки H1-H3: Inter Bold / Extra Bold, kerning +20
- Текст: Inter Regular / Medium, kerning 0
- Мелкий текст: Inter Light, kerning +40

Акцентный шрифт (логотип, цифры, бейджи): Manrope или Montserrat
- Использовать только для крупных элементов

Моноширинный шрифт (код, настройки, технические элементы): JetBrains Mono

Размеры шрифтов (rem = 16px):
- H1: 36-48px / Bold / kerning +20
- H2: 24-30px / SemiBold / kerning +10
- H3: 20-24px / Medium / kerning 0
- Основной текст: 16px / Regular / kerning 0
- Мелкий текст: 12-14px / Light / kerning +40
- Подписи: 10-11px / Mono / kerning +20

Трекинг (letter-spacing) по умолчанию:
- Заголовки: -0.02em
- Текст: 0
- Подписи: +0.05em
- Технические элементы: +0.1em


5. ТЕНИ И ГЛОУ
--------------
Тени для карточек:
- Светлая тема: 0px 2px 8px rgba(0,0,0,0.04)
- Тёмная тема: 0px 4px 12px rgba(0,0,0,0.3)

Тени для кнопок (при ховере):
- box-shadow: 0px 0px 12px rgba(76,175,80,0.4)

Глоу эффекты:
- Успех: 0px 0px 16px rgba(76,175,80,0.3)
- Золотой акцент: 0px 0px 20px rgba(255,167,38,0.2)


6. КНОПКИ И ИНТЕРАКТИВНЫЕ ЭЛЕМЕНТЫ
----------------------------------
Primary кнопка:
- Фон: градиент 1 (FarmNight → FarmSpark)
- Текст: белый, 16px, Medium
- Скругление: 12px
- Паддинг: 12px 24px
- Ховер: смещение градиента + поднятие (translateY -2px)
- Актив: scale 0.98

Secondary кнопка:
- Фон: прозрачный, граница white/0.2
- Текст: белый
- Скругление: 12px
- Ховер: фон white/0.05, граница white/0.4

Ghost кнопка:
- Фон: прозрачный
- Текст: FarmSpark
- Без границы
- Ховер: underline + opacity 0.8


7. АНИМАЦИИ
-----------
Длительности по умолчанию:
- Быстрые (hover, фокус): 0.15s
- Средние (появление, закрытие): 0.3s
- Медленные (страницы, переходы): 0.5s

Easing функции:
- Упругий вход: cubic-bezier(0.34, 1.2, 0.64, 1)
- Стандартный: ease-in-out
- Плавный: cubic-bezier(0.4, 0, 0.2, 1)

Стандартные анимации:
- Появление блока: fade in + translateY(10px) / 0.3s
- Ховер кнопки: translateY(-2px) + shadow / 0.15s
- Загрузка: пульсация или микро-точки (3 точки с задержкой)
- Рост (особая анимация): росток, поднимающийся вверх, листья раскрываются

Микро-анимации:
- Переключение темы: плавное изменение цвета фона (0.3s)
- Успешное действие: вспышка зелёным (0.2s) + возврат
- Ошибка: горизонтальное покачивание (0.3s)


8. ИКОНКИ
---------
Стиль: линейные с заливкой градиента на отдельных элементах
Толщина линии: 1.5-2px
Цвет: FarmSpark или белый (в зависимости от фона)
Скругление углов: 2-4px

Кастомные иконки:
- Логотип: росток в круге или буква "П" в форме колоска
- Постинг: лист с текстом
- Синхронизация: два круга с бегущей точкой между ними
- Настройки: шестерёнка
- Аналитика: столбик из трёх элементов разной высоты (как колосья)
- Помощь: круг с вопросительным знаком

Сторонняя библиотека: react-icons (Fa, Hi, Cg, Md)

------------------------

- Текст второстепенный: #7F8D7F

- Тени: глубокие, 0px 4px 12px rgba(0,0,0,0.3)

Переключение темы: плавное (0.3s), через CSS-переменные


10. ОТСТУПЫ И СЕТКА
-------------------
Базовый отступ: 8px (step)
Стандартные отступы: 4, 8, 12, 16, 24, 32, 48, 64px

Сетка:
- Контейнер: max-width 1280px, padding 24px
- Колонки: 12-колоночная сетка
- Гуттер: 24px

Скругления:
- Маленькие элементы (иконки, бейджи): 4px
- Средние (карточки, инпуты): 8px
- Большие (модалки): 12px
- Круглые (аватарки): 50%


11. ТИПОВЫЕ ЭКРАНЫ
------------------
Главный экран конструктора:
- Шапка: логотип ПостФерма слева, аватар пользователя справа
- Приветствие: "Привет, [имя]. Пора сеять контент!"
- Блок выбора платформы: карточки Telegram и VK с анимацией синхронизации
- Блок-конструктор: список блоков (Текст, Клавиатура, Рассылка, AI)
- Кнопка "Опубликовать": градиентная, крупная

Экран загрузки:
- Логотип ПостФерма по центру
- Три пульсирующие точки под логотипом
- Фон: тёмно-зелёный (#1E3A1E)

Экран ошибки:
- Иконка ошибки (красный круг с крестом)
- Сообщение об ошибке (белый текст, 16px)
- Кнопка "Повторить" (Secondary)
- Фон: тёмно-зелёный


12. ЗВУКИ (опционально)
-----------------------
Успешное действие: короткий "тинг" (колокольчик)
Публикация поста: шелест листьев
Ошибка: мягкий "пуф" (не раздражающий)
Синхронизация: звук роста + "динь" в конце


13. КЛЮЧЕВЫЕ КОМПОНЕНТЫ
-----------------------
- Logo — иконка (росток/колос) + текст "ПостФерма" + загрузка
- Button — primary / secondary / ghost
- Card — карточка с градиентной границей при ховере
- Input — поле ввода с микро-анимацией фокуса
- Modal — модальное окно с blur-фоном
- Toast — уведомление (успех / ошибка / информация)
- Loader — три пульсирующие точки
- Toggle — переключатель темы
